# JOGO DA FORCA
Jogo da forca em C, utilizando alocação dinâmica e arquivo.

### LICENÇA
[Leia a licença](LICENSE) **MIT**


### CARACTERÍSTICAS:
- Possibilita a utilização de palavras de tamanho qualquer
- Pega as palavras em arquivo
- Pega as dicas em arquivo
- MVC


### TESTADO NO:
- Ubuntu 16.04
